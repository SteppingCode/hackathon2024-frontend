SvelteKit • Tailwind • FastAPI

Современная веб-панель для соревнований и пользовательских сценариев.

FindSport Frontend — это аккуратный интерфейс для управления заявками, соревнованиями, федерациями, административными задачами и профилями пользователей. Проект собран на SvelteKit с современным UI и удобной интеграцией с backend.

Посмотреть возможности Технологический стек

Что реализовано

Соревнования + UI
Заявки + workflow
Админка + сценарии
Профиль + настройки

Ключевые возможности

Удобная работа с соревнованиями

Интерфейс с карточками, списками и диалогами помогает быстро ориентироваться в заявках и соревнованиях.

Авторизация и профиль

Пользовательские сценарии, настройки профиля и понятный путь взаимодействия с системой.

Административные сценарии

Поддержка задач по работе с федерациями, управлению пользователями и другими операциями.

Современный UX

Визуальный стиль основан на Tailwind и UI-компонентах для быстрых и приятных взаимодействий.

Технологический стек

SvelteKit Svelte 5 Tailwind CSS shadcn/ui Axios Lucide Svelte ESLint Prettier

Локальный запуск

Для локальной разработки понадобится Node.js 18+ и pnpm или npm. После установки зависимостей приложение можно запустить в режиме разработки.

pnpm install
pnpm dev
